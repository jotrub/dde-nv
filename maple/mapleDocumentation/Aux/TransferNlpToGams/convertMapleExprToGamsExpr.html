@path

 Aux/TranferNlpToGams/convertMapleExprToGamsExpr.txt

@brief

 Converts maple expression to expression with appropriate GAMS grammar

@param

 Maple expression (e.g. a^2+b)

@return

 String with the rewritten expression according by GAMS syntaxis

@authors

 dka

@notes

 (1) The input parameter (maple expression) is not validated
        (2) Rules for fransformation: a^2 -> sqr(a)
                                      a^(1/2) -> sqrt(a)
                                      a^(-b) -> 1/a^b
                                      a^b -> power(a,b), if b is an integer number         
                                      a^b -> exp(a,b), if b is an integer number
